Anforderungen

Die Programme der Asklaion Projekte haben unterschiedliche Systemanforderungen. Hier eine Übersicht der Anforderungsstufen:

WEBAPP

Web-Applikation

Keine Installation notwendig. Läuft direkt im Browser – empfohlen wird ein aktueller Browser (Chrome, Firefox, Edge, Safari). Es gibt keine besonderen Hardwareanforderungen, da die gesamte Verarbeitung auf dem Server stattfindet.

GERING

Geringe Anforderungen

Es werden keine wirklichen Voraussetzungen an den Rechner gestellt. Ein handelsüblicher PC oder Laptop mit etwas normalem Arbeitsspeicher (RAM) reicht vollkommen aus. Es wird keine besondere Grafikkarte benötigt.

  • Windows 10/11 oder macOS 12+
  • 4 GB RAM
  • Keine besondere Grafikkarte notwendig
  • 500 MB freier Speicherplatz
MITTEL

Mittlere Anforderungen

Es wird ein Rechner mit mindestens einer Grafikkarte benötigt, um die lokalen KI-Modelle auszuführen. Eine integrierte oder einfache dedizierte GPU ist hierfür ausreichend.

  • Windows 10/11 oder macOS 12+
  • 8 GB RAM
  • Integrierte oder einfache dedizierte Grafikkarte ausreichend
  • 2 GB freier Speicherplatz
HOCH

Hohe Anforderungen

Es wird ein leistungsstarker Rechner mit mindestens einer guten dedizierten Grafikkarte benötigt. Die KI-Modelle laufen lokal und brauchen entsprechend Rechenleistung, um performant zu arbeiten.

  • Windows 10/11 oder macOS 12+
  • 16 GB RAM (32 GB empfohlen)
  • Dedizierte GPU mit min. 8 GB VRAM empfohlen
  • 10 GB freier Speicherplatz
  • Für KI-Modelle: NVIDIA GPU mit CUDA-Support
