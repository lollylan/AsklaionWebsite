# Dein erster Prompt

In dieser Lektion lernst du, wie du der KI klare Anweisungen gibst, damit sie genau das Programm erstellt, das du dir vorstellst.

## Was ist ein Prompt?

Ein Prompt ist eine **Anweisung in natürlicher Sprache**, die du der KI gibst. Je klarer und strukturierter dein Prompt, desto besser das Ergebnis.

## Die Anatomie eines guten Prompts

Ein guter Prompt enthält:

1. **Kontext** – Was ist das Ziel?
2. **Spezifikation** – Was genau soll das Programm tun?
3. **Einschränkungen** – Welche Technologien oder Grenzen gibt es?

### Beispiel

```
Erstelle mir eine einfache Webseite mit einer Stoppuhr.
Die Stoppuhr soll Start, Stopp und Reset-Buttons haben.
Verwende HTML, CSS und JavaScript.
Das Design soll modern und dunkel sein.
```

## Übung

Versuche selbst, einen Prompt für folgendes Programm zu schreiben:

> **Aufgabe:** Erstelle einen Prompt für einen einfachen Taschenrechner, der die vier Grundrechenarten beherrscht.

## Zusammenfassung

- Prompts sind dein wichtigstes Werkzeug beim Vibecoding
- Klarheit und Struktur führen zu besseren Ergebnissen
- Übung macht den Meister – je mehr Prompts du schreibst, desto besser wirst du
